-= RailsCms =-

Работа с RailsCms 'с нуля'

1. Создаем новый RVM (rails 3.2.*)
2. Создаем новый Rails-проект
3. Добавляем DevCms-плагин, подключаем его как GEM-файл
4. Прогоняем bundle для установки всех зависимостей
5. Запускаем инсталлятор = rake devcms:setup 
5. Запускаем сервер
6. Входим в админку, создаем страницу и редактируем ее.
